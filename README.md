## Koma in one glance

- Built on Mihon + TachiyomiSY, focused on fast navigation and readable pages.
- Signature *Midnight Deep* theme and floating island navigation.
- Global search with filters, exclusions, and saved queries.
- Migration tools and batch actions for organizing large libraries.

---

## Feature map

### Reading flow
- Multiple modes: continuous, paged, webtoon, auto-scroll
- Smart backgrounds that adapt to page content
- Webtoon optimization with forced zoom control
- Offline reading with flexible download caching

### Library control
- Drag-and-drop categories with custom grouping
- Edit covers, notes, and metadata
- Batch operations: clean titles, merge duplicates, mass import
- Quick actions for bookmark, download, and updates

### Discovery & tracking
- Global search with quotes, exclusions, and saved searches
- Sync progress with MyAnimeList, AniList, Kitsu, MangaUpdates, Shikimori, Bangumi
- Manga recommendations via MAL, AniList, and related titles

### Source power-ups
- Built-in enhancements for E-Hentai / ExHentai, Mangadex, 8Muses, NHentai, HBrowse, Puruin, LANraragi
- Seamless source migration with intelligent matching

### Design touches
- Midnight Deep theme plus full theming support
- Floating island navigation with smooth animation
